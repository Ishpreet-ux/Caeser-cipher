# Caesar Cipher Web App

A simple and interactive web-based Caesar Cipher encryptor and decryptor built using HTML, CSS, and JavaScript.

---

## About the Project

This project demonstrates the working of the Caesar Cipher, one of the oldest and simplest encryption techniques in cryptography.

Users can:

* Encrypt messages
* Decrypt messages
* Enter custom shift values
* Use the app directly in a web browser

---

## Features

* Real-time text encryption and decryption
* Custom shift value support
* Clean and responsive UI
* Beginner-friendly JavaScript logic
* Lightweight and fast

## How Caesar Cipher Works

The Caesar Cipher shifts each letter in a message by a fixed number of positions in the alphabet.

Example:

```text
Original: HELLO
Shift: 3
Encrypted: KHOOR
```
